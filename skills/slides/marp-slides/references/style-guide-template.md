# スタイルガイド

このプロジェクトのスライドで使う配色・タイポ・文体のルール。新しいデッキやパターンを作る前に確認する。
色・タイポの実装（Tailwind config）は `templates/slide-header.md` に一元化されており、そちらが実装上の正。本ガイドは運用ルールとしての一次情報であり、`slide-header.md` を変更したら本ガイドも合わせて直す（詳細は「6. ブランド差し替え手順」）。

## 1. カラーパレット

グレースケール基調に、アクセント2色（濃紺・ティール）と補助1色（ルビー）を添える配色。

### グレースケール（`gray-50`〜`gray-900`）

背景・パネル濃淡・本文・見出しの4役割を、明度の段階に沿って割り当てる。

| クラス | 役割 |
| --- | --- |
| `gray-50` | ページ背景 |
| `gray-100` | パネル背景（明） |
| `gray-200` | 罫線・区切り線 |
| `gray-300` | パネル背景（暗め）・非活性表示 |
| `gray-400` | 補助テキスト（弱） |
| `gray-500` | キャプション・注釈 |
| `gray-600` | 本文中の強調（1段階目） |
| `gray-700` | 本文の基本色／強めの強調 |
| `gray-800` | 見出し |
| `gray-900` | タイトル・最重要見出し |

### アクセント2色 + 補助1色

| トークン | 値 | 用途 |
| --- | --- | --- |
| `brand-navy` | `#1B4565` | 主アクセント。タイトル・キーライン・強調パネル |
| `brand-teal` | `#3E9BA4` | 副アクセント。navy と対にして使う既定ペア |
| `brand-ruby` | `#B23A6B` | 補助アクセント。ここぞという1点強調のみに使い、多用しない |

### グラデーション

セクション見出しやヒーローパネルの背景には、navy→teal の対角グラデーションを既定にする。

```html
<div class="bg-gradient-to-br from-brand-navy to-brand-teal ...">
```

## 2. 運用ルール

- 1スライドで使うアクセント色は**1〜2色まで**（既定は navy＋teal のペア、ruby は1点強調の補助として足す場合のみ）。3色以上を同一スライドに混在させない。
- 強調表現に `red-*` / `green-*` は使わない。エラー・成功のような信号色として誤読されるため、濃淡による強弱で表現する `gray-600` / `gray-700` を使う。

## 3. タイポグラフィ（em クラス）

サイズ・行間の実体は `templates/slide-header.md` の Tailwind config（`theme.extend.fontSize`）にある。値を変える場合は必ず `slide-header.md` 側を直し、この表を実装値に合わせる（この表ではなく `slide-header.md` が正）。

| クラス | サイズ/行間 | 用途 |
| --- | --- | --- |
| `text-em-3xl` | 4.5rem / 1 | 大きな数値強調 |
| `text-em-2xl` | 2rem / 1.2 | パネル/セクション見出し |
| `text-em-xl` | 1.5rem / 1.3 | サブ見出し |
| `text-em-lg` | 1.25rem / 1.6 | 本文（強調） |
| `text-em-base` | 1rem / 1.7 | 本文 |
| `text-em-sm` | 0.85rem / 1.6 | 注釈・キャプション |

## 4. フォントスタック

日本語表示のレンダリング環境差を抑えるため、フォントは明示的に指定する（`slide-header.md` の `fontFamily.sans` と同じ値）。

```
"Hiragino Kaku Gothic ProN", "Yu Gothic", "Noto Sans JP", sans-serif
```

## 5. 文体ルール

- タイトルに**コロンを使わない**（「〜について:」のような体言止め＋コロンの表記をしない）。
- 感嘆符・疑問符・装飾絵文字を避け、理知的で落ち着いた文体にする。
- 色による強調は控えめにする。地の文は `gray-700` を基本とし、強調したい場合も色数を増やすのではなく濃淡（`gray-600`/`gray-700`）で表現する。

## 6. ブランド差し替え手順

プロジェクト専用の色に差し替える場合は、次の2箇所を必ずセットで直す（片方だけ直すと表示と本ガイドの記載がずれる）。

1. `templates/slide-header.md` の Tailwind config `colors.brand`（`navy` / `teal` / `ruby` の値）を書き換える。
2. 本ガイドの「1. カラーパレット」にある色名・値の記載を、変更後の値に合わせて更新する。
